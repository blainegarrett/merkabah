Modal

Modal is declared just like bootstrap modal, except it allows you to bind an ng-model to the modal, which is true/false depending on if the modal is open.


You can either open and close the modal the old bootstrap way (data-toggle/data-dismiss and href), or by setting the ng-model to true/false


The ng-model will always represent whether the modal is currently open or not.

$scope.modalShown = {{modalShown}}

Popover


This is a bit different from bootstrap's normal popover.

Instead of a simple title and content, you can have anything you want inside the popover.

The popover is fully controlled by attributes from the popup invoker element: the placement, the event, even the spacing.
You can do anything you want inside the popover? Cool!
